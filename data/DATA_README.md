# Data Source Documentation

## Raw Dataset Information

### Dataset Name
IBM HR Analytics Attrition Dataset

### Dataset Description
This analysis uses the IBM HR Analytics Attrition Dataset from Kaggle, which contains comprehensive employee information including demographics, job details, satisfaction metrics, and attrition status.

### Key Data Fields
The dataset includes the following columns:
- **Attrition**: Whether the employee left the company (Yes/No)
- **Department**: Employee's department (Sales, Research & Development, Human Resources)
- **JobSatisfaction**: Employee job satisfaction rating
- **MonthlyIncome**: Employee's monthly income in dollars

### How to Retrieve the Dataset

**Dataset Source**: [IBM HR Analytics Attrition Dataset on Kaggle](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset/data)

**Steps to Access**:
1. Visit the Kaggle dataset page: https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset/data
2. Sign in to your Kaggle account (or create a free account if you don't have one)
3. Click the "Download" button to obtain the dataset
4. Extract the CSV file from the downloaded archive

The data contains employee records with various attributes used to analyze attrition patterns, particularly in the Sales Department.

**Note**: The actual raw data file is not included in this repository to reduce repository size. Please download it directly from the Kaggle source above.

### Data Format
- **File Type**: Structured database table
- **Table Name**: `employee`
- **Approximate Size**: Contains employee records across multiple departments

### Data Usage
This dataset is used for analyzing employee attrition patterns, with a specific focus on the Sales Department to identify correlations between job satisfaction, monthly income, and employee retention.
